.. _commands_micromamba/config/describe:

``describe``
============

Usage: ``micromamba config describe [OPTIONS] [configs...]``


**Positionals:**

``configs TEXT ...``

Configuration keys.


**Options:**

``-h,--help``

Print this help message and exit.

``-l,--long-descriptions``

Display configs long descriptions.

``-g,--groups``

Display configs groups.
